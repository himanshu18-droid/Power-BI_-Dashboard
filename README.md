
🏥 Hospital Data Analysis Dashboard
📌 Project Overview

This project focuses on analyzing hospital data to extract meaningful insights and improve decision-making in healthcare. The dashboard provides a clear visualization of patient records, treatment trends, and hospital performance metrics.

🎯 Objectives
Analyze patient data efficiently
Identify trends in diseases and treatments
Improve hospital resource management
Create interactive and easy-to-understand dashboards

🛠️ Tools & Technologies Used
Power BI – Data visualization & dashboard creation
Microsoft Excel – Data cleaning & preprocessing
Basic Data Analysis Techniques

📊 Key Features
Patient admission analysis
Disease-wise data visualization
Department performance tracking
Interactive filters and slicers
Clean and user-friendly dashboard design
